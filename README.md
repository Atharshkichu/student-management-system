#  Student Management System

A web-based **Student Management System** built using **Django** and **MySQL**.  
This project allows users to manage student records with full CRUD operations, search, pagination, and authentication.

---

##  Features

-  User Authentication (Login & Logout)
-  Add Student
-  Edit Student
-  Delete Student
-  Search students by Name / Email / Department
-  Pagination for student list
-  Clean UI using Bootstrap + Custom CSS

---

##  Tech Stack

- **Backend:** Python, Django
- **Database:** MySQL
- **Frontend:** HTML, CSS, Bootstrap
- **Version Control:** Git & GitHub
